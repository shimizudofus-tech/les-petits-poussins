# Voix à générer (IA — voix femme, français)

Génère un MP3 par ligne ci-dessous, **voix féminine douce FR**, débit lent (enfants).
Nom de fichier = exactement la colonne **Fichier** (minuscules, sans accent).
Dépose-les dans `public/audio/voix/`. L'app les utilisera automatiquement (sinon voix navigateur).

Réglages conseillés : ~0,9x vitesse, ton chaleureux/joyeux.

## Génération automatique (OpenAI TTS)
Sur ta machine (réseau requis, clé API OpenAI — pas ChatGPT Plus) :
```powershell
cd petits-poussins
$env:OPENAI_API_KEY="sk-xxxx"
node scripts/generate-voices.mjs
```
Génère uniquement les manquants, voix `shimmer` (féminine), dans `public/audio/voix/`.
Changer la voix : `$env:VOICE="nova"`. Tout régénérer : `$env:FORCE="1"`.
Coût ~quelques centimes. Ensuite : `npm run build` puis commit/deploy.
